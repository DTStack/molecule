Molecule

A lightweight Web IDE UI framework

Quick Start
Preview

Easy to Use

The Molecule has a majority of built-in components and services which could combine these as your like, and achieve the complex interactive through event-emit.

Extensible

The Molecule could enrich the functionality through extensions. Besides, the Molecule also support some extensions from VSCode extensions market.

Powered by React

The Molecule a UI framework which is developed in React and associated with MVC pattern. It only export ES modules for using in React Project.
